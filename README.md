# Portfolio — SHAIK MOHAMMED AKRAM ALI

Simple single-page portfolio for Shaik Mohammed Akram Ali.

To publish:
1. Add all files (index.html, styles.css, script.js, README.md) to a Git repository.
2. Push to GitHub (e.g., https://github.com/SkAkramali/portfolio).
3. On GitHub: Settings → Pages → Source: select branch "main" and folder "/" (root). Save and wait a few minutes — your site will be available at https://SkAkramali.github.io/portfolio/.
